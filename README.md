# ProcessWatcher
このgem `heroku logs -t` を実行していると(おそらく)heroku側の影響でたまに終了したり、何も出力しない、ということがあるのでそれらが起きた時に再実行するためのものです。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'process_watcher'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install process_watcher
